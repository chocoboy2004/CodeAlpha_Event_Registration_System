# Event Registration System

## Project Description
The Event Registration System is built using Node.js, Express.js, MongoDB, Mongoose ODM, JWT, and Bcrypt. This system includes three different controllers:

1. **User Controller**: 
    - Signup
    - Signin
    - Signout
    - Update Name
    - Update Email
    - Update Password
    - Update Phone Number

2. **Event Controller**: 
    - Register Event
    - Delete Event
    - List Events
    - Get Event
    - Update Event Name
    - Update Event Description

3. **Registration Controller**: 
    - Register Event
    - Toggle Payment Status
    - Cancel Registration
    - List Registrations

## Technologies Used
- Node.js
- Express.js
- MongoDB
- Mongoose ODM
- JsonWebToken (JWT)
- Bcrypt
